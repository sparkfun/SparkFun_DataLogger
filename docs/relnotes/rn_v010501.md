
## SparkFun DataLogger IoT Firmware - Version 1.5.1

June 2026

### New Features and Enhancements

#### New Device Support

Support for the following new SparkFun qwiic devices

- [SparkFun Spectral Sensor - AS7343 (Qwiic)](https://www.sparkfun.com/sparkfun-spectral-sensor-as7343-qwiic.html)
- [Garmin LIDAR-Lite v4 LED - Distance Measurement Sensor (Qwiic) sensor](https://www.sparkfun.com/garmin-lidar-lite-v4-led-distance-measurement-sensor-qwiic.html)
- [SparkFun Ambient Light Sensor - VEML6030 (Qwiic) sensor](https://www.sparkfun.com/sparkfun-ambient-light-sensor-veml6030-qwiic.html)
- [SparkFun Spectral UV Sensor - AS7331 (Qwiic) sensor](https://www.sparkfun.com/sparkfun-spectral-uv-sensor-as7331-qwiic.html)

### General Improvements
* Fixed issue with auto loading some I2C addresses for the MCP9600 Thermocouple Amp device. All addresses should now work correctly

And this new functionality is available today at the DataLogger [repo](https://github.com/sparkfun/SparkFun_DataLogger) as a beta release.  The update is free, available as a file uploaded via an SD Card.  Just select the “System Update” option within the DataLogger IoT menu system and select your desired upgrade option.
